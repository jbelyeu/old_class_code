How to check a project into npm

  1. Most of this tutorial comes from the book starting on page 49 (Chapter 3, Listing 3.1)
  2. Change your directory so that you are in the folder with the files for censorify
  3. Create an account at npmjs
  4. run adduser on the command line and enter the account information from npmjs.com 
    npm adduser
  5. Create a package.json file (help here ) file to include the repository information from github with the fields he included in the example (except "dependencies")
  6. Change the "name" to something sensible (capital letters not allowed).
  7. Add a "repository" field (follow the pattern from Table 3.2 on that same page)
  8. run "npm pack" on the command line to create a zip file
  9. run "npm publish" to publish at npmjs.com
  10. You can find lots of documentation for npm including information on how to update your project.
  11. If you screw up, you can unpublish the project, update the version number and publish it again.
  12. Now create a new directory in your home directory called test. cd into that directory and type "npm install [npm name]". This should install your code in the new directory.
